Smart Farming Alert System

Aim

To protect crops from animals using an automatic alert system.

Materials Required

  • IR Sensor
  • Buzzer
  • Battery
  • Wires
  • Cardboard Model

Working

When an animal enters the farm area, the sensor detects movement and the buzzer rings to alert the farmer.

Advantages

  • Protects crops
  • Low cost
  • Easy to build

Uses

Useful for farms, gardens, and agricultural fields.
